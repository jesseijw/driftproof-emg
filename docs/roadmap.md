# Roadmap

## Months 1-2

- Build simulation and replay pipeline.
- Define session schema.
- Implement preprocessing and classical EMG features.
- Train/evaluate first baseline models on synthetic sessions.

## Months 3-4

- Build indexed armband.
- Record first raw EMG sessions.
- Characterize placement repeatability and degradation.

## Months 5-6

- Implement drift scores and uncertainty calibration.
- Add replay tests and scorecards.

## Months 7-8

- Implement adaptation baselines.
- Compare against no-adaptation and full recalibration.

## Months 9-10

- Integrate one-motor gripper and force sensing.
- Run pilot task studies.

## Months 11-12

- Freeze protocol.
- Run main study if feasible.
- Complete participant-level analysis.

## Months 13-18

- Release dataset and hardware docs.
- Write manuscript.
- Evaluate patentability only after a specific technical mechanism is proven.
